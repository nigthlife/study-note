02 DIRB

DIRB 是一个 Web 内容扫描器。它是 kali linux 内置的工具，通过对 Web 服务器发起基于字典的攻击并分析响应来工作，但请记住它是内容扫描器而不是漏洞扫描器。 使用也很简单，在DIRB后面直接加目标域名即可。


03 dirsearch
