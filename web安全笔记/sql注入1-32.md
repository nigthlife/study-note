>           select * from tableName limit i,n
>           tableName：表名

>           n：为查询结果返回的数量
>           i与n之间使用英文逗号","隔开
>           limit n 等同于 limit 0,n
>           limit 0,1 默认0（i）就是从1开始
>           ```
